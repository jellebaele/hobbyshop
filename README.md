# Hobbyshop

## Features
- Register, Login
- Confirm registration, forgot password
- Create, get, update, and delete products
- Order products (without payment)
- Order confirmation
- Update orders
- Orders page
  -  My orders
  -  Orders that have to be processed by you
-  Profile page
-  Overview users page

## Stack
- Front-end: React with Redux
- Back-end: Nodejs, Mongoose, Redis

**Deployment: **
- Front-end: Netlify
- Back-end: Heroku
